Unit 7: Recursion

public void expR (int base, int exponent) {
    if (exponent == 0) return 1;
    return base * expR (base, exponent - 1);
}
public void sumR (int num) {
    if (num == 0) return 0;
    return num + sumR(num - 1);
}
To the right, we have two methods.
The expR() method returns the exponent of the base to the power of the exponent using recursion.
The sumR() method returns the sum from 1 to num inclusive.
