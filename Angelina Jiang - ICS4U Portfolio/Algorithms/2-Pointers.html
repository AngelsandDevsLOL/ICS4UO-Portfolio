An algorithm used a lot in competitive programming. A pointer is a variable that can traverse through an array, normally pointing at elements
This is useful when going through a sorted array, whenever an array inquiry asks you to locate two integers in an array that either directly or indirectly satisfy a given criteria.
Example Problem: We want to find the number of pairs in an array with a difference of 5.
How would you solve this? Have a for loop with two variables, or pointers, pointing at 2 different elements in the array and compare.
Pseudocode:

Check the difference between the array values at pointer1 and pointer2.
While pointer2 is not at the end or the difference between pointer1 and pointer2 is smaller than 5,
    If this difference is equal to 5, increase count
    If pointer2 is not at the end of the array, move pointer2 to the right.
    If this difference is smaller than 5 and pointer2 is not at the end of the array, then move pointer2 to the right.
    If this difference is larger than 5, then move pointer1 to the right.
    Else break // this statement means that the difference is smaller than 5 and pointer2 is at the end of the array
    Update difference
int[] arr = {1, 2, 3, 4, 5, 6};
int pointer1 = 0;
int pointer2 = 1;
int count = 0;
int diff = arr[pointer2] - arr[pointer1];
while (pointer2 != arr.length - 1 || diff < 5) {
    if (diff == 5) {
        count++;
        if (pointer2 != arr.length - 1) pointer2++;
    } else if (diff < 5 && pointer2 != arr.length - 1) {
        pointer2++;
    } else if (diff > 5) {
        pointer1++;
    } else {
        break;
    }
    diff = arr[pointer2] - arr[pointer1];
}
System.out.println(count);

Even nested for loops can be used as pointers! If we had a for loop with variable i and another for loop with variable j, i and j would be pointers!
Two pointers are used a lot in competitive programming. They can be used to easily remove duplicates from a sorted array, rotate arrays, reverse words, etc.
